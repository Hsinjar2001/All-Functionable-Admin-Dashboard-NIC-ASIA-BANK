# NIC Bank Authentication API

FastAPI authentication backend with PostgreSQL and JWT.

## Features

- ✅ User Registration
- ✅ User Login (JWT)
- ✅ Get Current User
- ✅ Token Verification
- ✅ Profile Update
- ✅ Password Hashing (bcrypt)
- ✅ PostgreSQL Database
- ✅ Swagger Docs
